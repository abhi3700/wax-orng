### Building and running contract unit tests

- Requirements
    - docker, installed and configured to run without sudo
    - make
      
- Build and test
    ```console

    # Start the development docker
    make dev-docker-start

    # Build
    make build

    # Run tests
    make test

    # Clean all and exit
    make clean
    exit
    ```
